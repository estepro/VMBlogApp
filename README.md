### To Download:

```
git clone https://github.com/estepro/VMBlogApp.git
```

Then:

```
npm install

or

yarn
```

To run the server only:

```
npm run server

or

yarn server
```

To run the client only:

```
npm run client

or

yarn client
```

To run both together:

```
npm run dev

or

yarn dev
```
